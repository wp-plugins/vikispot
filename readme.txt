== Description ==

VikiSpot let you embed related dynamic content, such as related images, videos, news, and blogs to your posts.

== Installation ==

Upload the VikiSpot plugin to your blog, Activate it, then add the new VikiSpot widget to the sidebar.
Configure the topic (or leave blank for post specific topic) and other options and save!

For setting post specific topic, add a custom attribute "vikispot" to your post while editing.
