== Description ==

VikiSpot let you embed dynamic content, such as related images, videos, news, and blogs to your posts.

For a live demo, please visit:
http://about.vikispot.com/wordpress/dynamic-content/

For html markup to directly embed content to your post or template, please visit:
http://www.vikispot.com


== Installation ==

Upload the VikiSpot plugin to your blog, Activate it, then add the new VikiSpot widget to the sidebar.
Configure the topic (or leave blank for post specific topic) and other options and save!

For setting post specific topic, add a custom attribute "vikispot" to your post while editing.
